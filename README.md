# Penetration Testing of a Simulated Healthcare Network.
Overview:
This project simulates a real-world penetration test on a healthcare network to identify, exploit, and assess critical vulnerabilities across systems and services.

Objectives:
- Perform reconnaissance and network scanning.
- Identify vulnerabilities in exposed services.
- Exploit misconfigurations and known CVEs.
- Gain unauthorized access to systems.
- Provide remediation strategies.

Tools Used:
- Kali Linux.
- Nmap.
- Wireshark.
- Metasploit.
- Nikto.
- SEToolkit

The lab environment consisted of three virtual machines:
- Kali Linux (Attacker).
- Metasploitable 2 (Linux Server).
- Windows 10 (Workstation).
